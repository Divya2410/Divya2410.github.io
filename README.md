# Divya Mishra - Personal Portfolio Website

Welcome to my personal portfolio website! This site showcases my research, academic work, community involvement, and professional achievements in astronomy.

## 🌟 About This Site

This is a professional portfolio website built with HTML5 and CSS3, featuring:
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth modal-based page transitions
- **Portfolio Showcase**: Highlights of research projects, publications, and presentations
- **Professional Appearance**: Clean, modern design for academic and professional use
